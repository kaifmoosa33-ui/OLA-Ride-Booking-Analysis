# 🚖 OLA Ride Booking Analysis

## 📌 Project Overview

This project analyzes over **100,000 OLA ride bookings** using **MySQL** and **Power BI** to uncover insights into booking trends, revenue, cancellations, vehicle performance, and customer & driver ratings.

The project demonstrates the complete data analytics workflow from data cleaning and SQL analysis to building an interactive business dashboard.

---

## 🛠️ Tools Used

- Microsoft Excel
- MySQL
- Power BI

---

## 📂 Dataset

The dataset contains ride booking information including:

- Booking ID
- Customer ID
- Vehicle Type
- Booking Status
- Ride Distance
- Booking Value
- Payment Method
- Customer Rating
- Driver Rating
- Cancellation Reasons

---

## 📊 SQL Analysis

Business questions solved include:

- Retrieve successful bookings
- Average ride distance by vehicle type
- Customer cancellation analysis
- Driver cancellation analysis
- Top 5 customers
- Payment method analysis
- Revenue analysis
- Vehicle-wise ratings
- Incomplete rides analysis

The project includes both the SQL script and a PDF containing the solved SQL queries.

---

## 📈 Power BI Dashboard

The dashboard consists of **5 interactive pages**:

### 📍 Overall
- Total Bookings
- Total Booking Value
- Booking Status Breakdown
- Ride Volume Over Time

### 🚗 Vehicle Type
- Booking Value by Vehicle
- Successful Booking Value
- Average Distance Travelled
- Total Distance Travelled

### 💰 Revenue
- Revenue by Payment Method
- Daily Ride Distance
- Top 5 Customers

### ❌ Cancellation
- Total Bookings
- Successful Bookings
- Cancelled Bookings
- Cancellation Rate
- Customer Cancellation Reasons
- Driver Cancellation Reasons

### ⭐ Ratings
- Average Driver Ratings
- Average Customer Ratings
- Vehicle-wise Rating Comparison
